## 🚀 Мои проекты

1. Ручное и API-тестирование онлайн-кинотеатра Кинопоиск:
    - создала тест-план;
    - написала чит-листы, чек-листы и тест-кейсы для ручного тестирования и API - тестирования;
    - провела smoke, функциональное и нефункциональное тестирование;

    - оформила отчет о готовности системы и рекомендации по улучшению.

    - создала автотесты на Python для API и UI-тестирования;
    - описала классы и тесты с помощью декораторов Allure;
    - создала файл run.sh для автоматизации рутины при создании отчета в Allure;
    - провела тестирование и вывела отчет в Allure.
